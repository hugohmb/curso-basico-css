Media Queries são um recurso do CSS que permite aplicar estilos condicionalmente, dependendo das características do dispositivo ou da tela, como largura, altura, orientação, resolução, entre outras.


Elas são fundamentais no design responsivo, ou seja, no desenvolvimento de layouts que se adaptam bem a diferentes tamanhos de tela (celular, tablet, desktop etc.).


Conceito Básico

A sintaxe básica de uma Media Query é:

@media condição {
        /* CSS que será aplicado se a condição for verdadeira */
      }

Exemplo Prático

Veja um exemplo em que o fundo da página muda dependendo da largura da tela:

/* Estilo padrão (desktop) */
      body {
        background: white;
      }
      
      /* Se a largura da tela for de até 768px (tablet ou celular), mude o fundo */
      @media (max-width: 768px) {
        body {
          background: lightgray;
        }
      }

Conceito Básico: Flexbox

O Flexbox (ou "Flexible Box Layout") é um modelo de layout unidimensional no CSS que facilita o alinhamento, a distribuição e o espaçamento entre elementos em um contêiner, mesmo quando seu tamanho é desconhecido ou dinâmico.

A ativação do Flexbox ocorre com:

display: flex;

Isso transforma os elementos filhos diretos do contêiner em itens flexíveis.

Propriedades mais comuns do contêiner (pai):

flex-direction — Define a direção dos itens (linha, coluna, etc). justify-content — Alinha os itens no eixo principal (horizontal por padrão). align-items — Alinha os itens no eixo transversal (vertical por padrão). flex-wrap — Permite que os itens quebrem linha se não couberem.

Propriedades comuns dos itens (filhos):

flex-grow — Controla quanto o item pode crescer. flex-shrink — Controla quanto o item pode encolher. flex-basis — Define o tamanho inicial do item. align-self — Permite sobrescrever o align-items individualmente.

Exemplo Prático:

Um contêiner com três caixas alinhadas no centro:

/* CSS */
      .container {
        display: flex;
        justify-content: center;
        align-items: center;
        height: 100vh;
      }
      
      .box {
        width: 100px;
        height: 100px;
        margin: 10px;
        background: steelblue;
      }
<!-- HTML -->
      <div class="container">
        <div class="box"></div>
        <div class="box"></div>
        <div class="box"></div>
      </div>
